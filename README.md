## MasterplanXP

![description](Assets/Images/logo-menu-combined3.svg)

MasterplanXP is an Adventure / Campaign Design Studio Application that can be used to create adventures and maintain campaigns for you game.  This effort is a cross-platform implementation that is very heavily inspired from the original Masterplan app designed by Andy Aiken.

MasterplanXP is currently being developed using the AvaloniaUI 11+ Platform, C#, and .NET 8.
